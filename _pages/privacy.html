The short version

This is a personal static website. I use privacy-friendly analytics that record aggregate visit data with no cookies and no personal identifiers. I don't run ads, I don't profile visitors, and I don't sell data. If that's all you needed to know, you're done.

Hosting

This site is hosted on GitHub Pages. GitHub may log standard server information (IP address, browser type, referring page) as part of operating their infrastructure. That's between you and GitHub. See their Privacy Statement for details.

Analytics

I use GoatCounter, an open-source privacy-friendly analytics service, to understand which pages people read and where traffic comes from. It records the page visited, browser family, screen size, referrer, and country (derived from IP). No tracking cookies are set.

GoatCounter stores only aggregate metrics. Source data such as IP addresses and User-Agent strings is computed into aggregates and discarded, and full database access cannot identify an individual visitor. Per GoatCounter's own documentation, this design does not require a GDPR consent banner. Their privacy policy covers their role as the service provider.

Third-party resources

The site loads fonts from Google Fonts and a CSS framework from a CDN. These services may collect limited technical data when serving those files. I don't control that, but I've chosen providers with reasonable privacy practices.

External links

Links to LinkedIn, GitHub, and other external sites are provided for convenience. I'm not responsible for their privacy practices.
